# User Journey Maps

## Journey 1: Research Pathway (Dr. Alisha Sharma)

### Scenario
Finding and citing Moura's work on design leadership in governance

### Journey Stages

```
Awareness → Discovery → Exploration → Action → Advocacy
    ↓           ↓            ↓           ↓          ↓
Citation → Search → Find Paper → Download → Share
```

### Detailed Flow

**1. Awareness** (External)
- Sees citation in journal article
- Notes author name and paper title
- Opens new browser tab
- Googles "Moura Quayle design leadership"

**2. Discovery** (Landing)
- Arrives at Living Archive homepage
- *Emotion*: Curious but skeptical
- *Need*: Quick credibility check
- Scans for academic indicators
- Sees "Living Archive" - intrigued

**3. Exploration** (Search)
- Uses search bar immediately
- Types partial paper title
- *Emotion*: Hopeful
- Sees auto-suggestions - relieved
- Clicks correct result

**4. Evaluation** (Paper Page)
- Checks publication details
- Verifies peer review status
- *Emotion*: Satisfied with credibility
- Notices related papers
- Sees download button

**5. Action** (Download)
- Clicks download
- PDF opens immediately
- *Emotion*: Pleased with efficiency
- Copies citation info
- Downloads 2 related papers

**6. Deeper Exploration**
- Returns to browse archive
- Filters by topic
- Discovers David's work too
- *Emotion*: Intellectually stimulated
- Bookmarks several items

**7. Advocacy**
- Shares link with research group
- Adds to bibliography
- Returns monthly for updates
- *Emotion*: Connected to community

### Pain Points & Solutions

| Pain Point | Solution |
|------------|----------|
| Can't find paper quickly | Smart search with auto-complete |
| Unclear if peer-reviewed | Clear publication badges |
| Paywall anxiety | Free download messaging |
| Lost after download | Related content prominent |
| No citation format | Copy button for citations |

## Journey 4: Citation/Download Flow

### Micro-Journey: Paper Access

```
Find → Verify → Access → Cite → Store
  3s      5s       2s     10s     5s
```

**Optimal Flow**:
1. **Find** (3 seconds)
   - Search or browse
   - Clear result display
   - Confidence in choice

2. **Verify** (5 seconds)
   - Check authors
   - Confirm date
   - See abstract

3. **Access** (2 seconds)
   - One-click download
   - Opens in browser
   - Option to save

4. **Cite** (10 seconds)
   - Copy formatted citation
   - Multiple format options
   - Export to tools

5. **Store** (5 seconds)
   - Save to device
   - Add to collection
   - Tag for later

**Total Time**: 25 seconds to completed citation
