# Homepage Concepts - Design Discovery

## Concept 1: "The Living Canvas"
**Visual Metaphor**: Watercolor wash that subtly animates

### Layout Structure
```
┌─────────────────────────────────────┐
│ Navigation Bar (minimal, floating)   │
├─────────────────────────────────────┤
│                                     │
│  "The Living Archive of             │
│   Moura Quayle & David Fusthey"     │
│                                     │
│  [Animated watercolor background]   │
│                                     │
│  Three entry points:                │
│  ┌─────┐ ┌─────┐ ┌─────┐          │
│  │Archive│ │Musings│ │Artworks│     │
│  └─────┘ └─────┘ └─────┘          │
│                                     │
├─────────────────────────────────────┤
│ Recent Additions (cards sliding in) │
└─────────────────────────────────────┘
```

**Key Features**:
- Watercolor texture animates subtly (breathing effect)
- Entry points have hover states pulling from artwork
- Recent additions show "living" nature
- Minimal text, maximum impact

**Rationale**: Immediately establishes artistic credentials while maintaining clarity

## Concept 2: "The Scholarly Garden"
**Visual Metaphor**: Organized growth with organic elements

### Layout Structure
```
┌─────────────────────────────────────┐
│ MQ | Navigation: About Archive Art  │
├─────────────────────────────────────┤
│ Hero: Split screen                  │
│ ┌─────────────┬─────────────┐      │
│ │ Text side   │ Visual side │      │
│ │ "Where      │ [Rotating   │      │
│ │ scholarship │  artwork]   │      │
│ │ meets art"  │             │      │
│ └─────────────┴─────────────┘      │
├─────────────────────────────────────┤
│ Search bar with smart suggestions   │
├─────────────────────────────────────┤
│ Featured sections (3-column grid)   │
│ ┌─────┐ ┌─────┐ ┌─────┐          │
│ │Latest│ │Popular│ │Collections│    │
│ └─────┘ └─────┘ └─────┘          │
└─────────────────────────────────────┘
```

**Key Features**:
- Split personality shown visually
- Smart search prominent
- Grid system for organization
- Academic meets artistic literally

**Rationale**: Balances both aspects of identity clearly

## Concept 4: "The Timeline River"
**Visual Metaphor**: Flowing stream of work

### Layout Structure
```
┌─────────────────────────────────────┐
│ The Living Archive | Search | Menu  │
├─────────────────────────────────────┤
│                                     │
│ Welcome statement (brief)           │
│                                     │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
│ 2024  2023  2022  2021  2020  ...  │
│ ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ │
│                                     │
│ [Scrolling timeline of works]       │
│ ┌───┐ ┌───┐ ┌───┐ ┌───┐          │
│ │Art│ │Paper│ │Talk│ │Musing│      │
│ └───┘ └───┘ └───┘ └───┘          │
│                                     │
└─────────────────────────────────────┘
```

**Key Features**:
- Chronological "river" of work
- Mixed media types in stream
- Infinite scroll reveals history
- Filter by type/topic/person

**Rationale**: Shows evolution and "living" nature temporally

## Concept 5: "The Reading Room"
**Visual Metaphor**: Inviting scholarly space

### Layout Structure
```
┌─────────────────────────────────────┐
│ [Subtle navigation bar]             │
├─────────────────────────────────────┤
│                                     │
│ Large welcoming headline            │
│ "Enter the Living Archive"          │
│                                     │
│ ┌─────────────────────────────┐    │
│ │ Featured work (large card)   │    │
│ │ [Changes periodically]       │    │
│ └─────────────────────────────┘    │
│                                     │
│ Quick links: About | Browse | Search│
│                                     │
├─────────────────────────────────────┤
│ Recent musings | Latest additions   │
└─────────────────────────────────────┘
```

**Key Features**:
- Book-like typography focus
- Featured work rotates
- Minimal, content-first
- Warm, inviting feeling

**Rationale**: Appeals to academic audience comfort

## Concept Comparison Matrix

| Concept | Strengths | Weaknesses | Best For |
|---------|-----------|------------|----------|
| Living Canvas | Artistic impact, memorable | May overwhelm some | First-time visitors |
| Scholarly Garden | Clear dual nature | More conventional | Researchers |
| Constellation | Unique, exploratory | Learning curve | Repeat visitors |
| Timeline River | Shows evolution | Linear thinking | Historians |
| Reading Room | Familiar, comfortable | Less distinctive | Academics |

## Recommendation for Testing
Test concepts 1, 2, and 3 with stakeholders as they represent the widest range of approaches while maintaining the "Bright Academia" aesthetic.
