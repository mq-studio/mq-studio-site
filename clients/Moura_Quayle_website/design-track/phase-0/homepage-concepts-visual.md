## Interaction Notes

### Living Canvas
- Watercolor background subtly shifts/breathes
- Entry cards have magnetic hover effect
- Recent additions slide in from left

### Scholarly Garden
- Artwork rotates every 10 seconds
- Hover on either side highlights connection
- Search bar expands on focus

### Constellation
- Nodes pulse gently
- Hover reveals connection paths
- Click reorganizes around selected node

### Timeline River
- Horizontal scroll through time
- Items float up slightly on hover
- Filter animates timeline compression

### Reading Room
- Featured work fades between selections
- Minimal animation for focus
- Smooth transitions between sections

These visual representations show how each concept balances the "Bright Academia" aesthetic with different approaches to information architecture and user engagement.
