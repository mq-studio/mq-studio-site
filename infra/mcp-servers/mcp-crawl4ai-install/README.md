# MCP Crawl4AI Installation

## Overview
This project provides the necessary setup for the Crawl4AI MCP server, which is designed to enhance AI capabilities through advanced crawling techniques. 

## Installation Instructions
To install the Crawl4AI MCP server, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/coleam00/mcp-crawl4ai-rag.git
   ```

2. Navigate to the project directory:
   ```bash
   cd mcp-crawl4ai-install
   ```

3. Run the installation script:
   ```bash
   ./scripts/install.sh
   ```

## Usage
After installation, the Crawl4AI MCP server can be started and configured as per the requirements outlined in the documentation.

## Dependencies
Ensure that you have the necessary dependencies installed as specified in the `install.sh` script.

Refer to the `docs/MCP_SERVER.md` file for detailed documentation on the Crawl4AI MCP server, including its purpose, configuration, and testing status.
