# Business Optimization Projects

This directory contains projects focused on optimizing business operations using IDP capabilities.

## Project Type: Business Optimization

**Purpose**: Enhance internal business processes and operational efficiency through automation and digital solutions.

**Examples**:
- Process automation workflows
- Business intelligence dashboards  
- CRM integration projects
- Document management systems
- Operational efficiency tools

## Directory Standards

- Each project gets its own subdirectory
- Follow the Project Taxonomy Standard for classification
- Include manifest.md in each project root
- Apply business-focused governance requirements

## Governance

Projects in this directory are automatically classified as **Business Optimization** type with appropriate:
- Internal deployment standards
- Business impact assessment requirements
- User acceptance testing protocols
- ROI measurement frameworks

See [Project Taxonomy Standard](../infra/dev-env-docs/PROJECT_TAXONOMY_STANDARD.md) for complete details.
